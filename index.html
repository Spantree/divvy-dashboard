Ride Dashboard

Divvy is a bike sharing system in Chicago that makes thousands of bikes available for rent at various locations throughout the city. Each station has a touchscreen kiosk and a docking system that releases bikes using a membership key or a ride code.

In early 2014, Divvy released a data feed of the nearly 760,000 rides taken on Divvy bikes in its inaugural year of 2013. This was part of the Divvy Data Challenge. Unfortunately, Spantree was heads down working on a few client projects during the challnege and didn't get a chance to compete, but we later took the opportunity to incorporate Divvy data into our Elasticsearch Workshop. We loved playing with the data so much that we decided to also create this data visualization demo powered by D3.js, Crossfilter, dc.js and Coffeescript. It works best in Chrome. All source code is available on Github.

The graphs you see below are entirely interactive. Data is loaded directly into the browser as a CSV and all the processing and filtering takes place completely in the browser as well. Some filters are still a little slow, but feel free to drill into the various dimensions and learn how Chicago fell in love with these little blue bicycles.
